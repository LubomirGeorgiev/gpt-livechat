# GPT-LIVECHAT

# Installation guide

1. `pnpm install`
2. `docker compose up -d`
3. Copy `.env.example` as `.env`
4. `pnpm run db:push`
5. `pnpm run dev`

## Tech Stack

- [Next.js](https://nextjs.org)
- [NextAuth.js](https://next-auth.js.org)
- [Prisma](https://prisma.io)
- [Drizzle](https://orm.drizzle.team)
- [Tailwind CSS](https://tailwindcss.com)
- [tRPC](https://trpc.io)
